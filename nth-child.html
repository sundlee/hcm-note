# nth-child

  • 특정 순서에 있는 요소를 선택할 때 사용하는 선택자입니다.
  • Mastering the :nth-child

# :nth-child(8)

:nth-child(8)를 통해서 8번째 엘레멘트를 선택할 수 있습니다.

li:nth-child(8) span {
    background-color: #298EB2;
    box-shadow: -3px -3px 10px rgba(0, 0, 0, 0.4), inset 0 0 10px black;
}

# nth-child(n+6)

6번째부터 끝까지의 엘리먼트를 선택할 수 있습니다.

li:nth-child(n+6) span {
    background-color: #298EB2;
    box-shadow: inset -3px -3px 10px rgba(0, 0, 0, 0.4), 0 0 10px black;
}

# nth-child(n+4):nth-child(-n+8)

4번째 ~ 8번째 엘리먼트를 선택할 수 있습니다.

li:nth-child(n+4):nth-child(-n+8) span {
    background-color: #298EB2;
    box-shadow: inset -3px -3px 10px rgba(0, 0, 0, 0.4), 0 0 10px black;
}

# nth-child(n+2):nth-child(odd):nth-child(-n+9)

2번째 ~ 9번째 엘리먼트 중 홀수번째의 엘리먼트를 선택할 수 있습니다.

li:nth-child(n+2):nth-child(odd):nth-child(-n+9) span {
    background-color: #298EB2;
    box-shadow: inset -3px -3px 10px rgba(0, 0, 0, 0.4), 0 0 10px black;
}
