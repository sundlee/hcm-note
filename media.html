# media query

media query는 장치에 따라 각기 다른 레이아웃을 구성하기위해서 사용하는 구문입니다.

/* small 디바이스용 */
@media screen and (max-width: 319px) {
  ...
} 

/* 모바일용 */
@media screen and (min-width: 320px) and (max-width: 999px) {
  ...
}

/* 데스크탑용 */ 
@media screen and (min-width: 1000px) {
  ...
}
