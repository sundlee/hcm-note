# flex

  • 웹이 표시되는 기기가 다양해지면서 CSS3에서는 element의 배치하는 방법으로 flex가 추가되었습니다. (Flexible Box Layout)
  • flex에 대한 설명
div { display : flex ; }

# flex direction

포함된 element의 배치 방향 

.container {
  flex-direction: row;
}

# flex-wrap

포함된 element의 줄바꿈을 허용할 것인지에 대한 정의 

.container{
  flex-wrap: wrap;
}

# justify-content

포함된 element의 정렬 방식을 정의

.container {
  justify-content: flex-end;
}
